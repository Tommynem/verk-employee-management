# Verk Extension: Employee Management

## Features

- Time tracking with gleitzeit (flexible hours) support
- Absence tracking (vacation, sick leave)
- Vacation day balance calculation (German law compliant)
- Monthly summaries and all-time balance calculation
- Data import/export functionality
- Settings management (tracking defaults, weekday patterns)

## Quick Start

```bash
# Install dependencies
make install

# Run development server
make dev

# Run tests
make test
```
